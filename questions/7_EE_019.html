Question 1Grade 7 • EE
A specific type of dust particle has a diameter of 3 x 10^-7 meters. If a scientist aligns 2 x 10^4 of these particles end-to-end, what is the total length in meters?
A. 6 x 10^-28 meters
B. 5 x 10^-3 meters
C. 6 x 10^-3 meters
D. 6 x 10^-11 meters
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
To find the total length when aligning particles end-to-end, we need to multiply the diameter of one particle by the number of particles.
The problem asks for a total length from multiple identical objects, which indicates multiplication.
2
(3 x 10^-7) x (2 x 10^4) = (3 x 2) x (10^-7 x 10^4) = 6 x (10^-7 x 10^4)
When multiplying numbers in scientific notation, multiply the numerical parts (coefficients) together.
3
6 x 10^(-7 + 4) = 6 x 10^-3
When multiplying powers of 10, add their exponents.
4
The total length is 6 x 10^-3 meters.
The coefficient (6) is between 1 and 10, so the number is in proper scientific notation.

Answer: 6 x 10^-3 meters (Option C)

E. None of the above

Step-by-step solution
1
To find the time it takes, we need to divide the total file size by the download speed.
The problem asks for a duration given a total amount and a rate, which implies division.
2
(3 x 10^6) / (9 x 10^8) = (3 / 9) x (10^6 / 10^8) = 0.333... x (10^6 / 10^8)
When dividing numbers in scientific notation, divide the numerical parts (coefficients) together.
3
0.333... x 10^(6 - 8) = 0.333... x 10^-2
When dividing powers of 10, subtract the exponent of the divisor from the exponent of the dividend.
4
The coefficient 0.333... is not between 1 and 10. Move the decimal point one place to the right to get 3.33.... To compensate, decrease the exponent by 1. 0.333... x 10^-2 = 3.33... x 10^(-2 - 1) = 3.33 x 10^-3
For proper scientific notation, the coefficient must be a number greater than or equal to 1 and less than 10.
5
It would take approximately 3.33 x 10^-3 seconds.
The final answer is in proper scientific notation.

Answer: 3.33 x 10^-3 seconds (Option C)

A. 2.0 x 10^2 months
B. 2.88 x 10^15 months

D. 2.0 x 10^15 months

Step-by-step solution
1
Total production = (Number of factories) x (Production per factory per month) = (3.0 x 10^1) x (4.0 x 10^5) Multiply coefficients: 3.0 x 4.0 = 12.0 Add exponents: 1 + 5 = 6 Result: 12.0 x 10^6 kg/month
First, find out how much steel all factories can produce collectively in one month using multiplication rules for scientific notation.
2
12.0 x 10^6 is not in proper scientific notation. Move the decimal one place to the left to get 1.2. To compensate, increase the exponent by 1. 12.0 x 10^6 = 1.2 x 10^(6 + 1) = 1.2 x 10^7 kg/month
Ensure the number is in proper scientific notation (coefficient between 1 and 10).
3
Months = (Total steel required) / (Total production per month) = (2.4 x 10^8) / (1.2 x 10^7) Divide coefficients: 2.4 / 1.2 = 2.0 Subtract exponents: 8 - 7 = 1 Result: 2.0 x 10^1 months
Divide the total steel needed by the monthly production capacity using division rules for scientific notation.
4
It will take 2.0 x 10^1 months (or 20 months).
The final answer is in proper scientific notation.

Question 10Grade 7 • EE
A popular athletic event attracted 2.1 x 10^6 spectators. Of these, 3.5 x 10^5 spectators bought VIP tickets, and the rest bought standard tickets. If the event venue has a capacity of 7.0 x 10^4, how many times the venue's capacity was filled by standard ticket holders? (Express the answer in scientific notation)
A. 2.5 x 10^3 times
B. 2.5 x 10^1 times
C. -2.0 x 10^1 times
D. 2.5 x 10^9 times
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Standard tickets = (Total spectators) - (VIP ticket holders) = (2.1 x 10^6) - (3.5 x 10^5) Align exponents: Convert 3.5 x 10^5 to 0.35 x 10^6 Subtract coefficients: 2.1 - 0.35 = 1.75 Result: 1.75 x 10^6 standard ticket holders
First, find the number of standard ticket holders by subtracting the VIP ticket holders from the total spectators. Remember to align exponents before subtracting.
2
Times filled = (Standard ticket holders) / (Venue capacity) = (1.75 x 10^6) / (7.0 x 10^4) Divide coefficients: 1.75 / 7.0 = 0.25 Subtract exponents: 6 - 4 = 2 Result: 0.25 x 10^2
Divide the number of standard ticket holders by the venue capacity using division rules for scientific notation.
3
0.25 x 10^2 is not in proper scientific notation. Move the decimal one place to the right to get 2.5. To compensate, decrease the exponent by 1. 0.25 x 10^2 = 2.5 x 10^(2 - 1) = 2.5 x 10^1
Ensure the final number is in proper scientific notation (coefficient between 1 and 10).
4
The venue's capacity was filled 2.5 x 10^1 times by standard ticket holders.
The final answer is in proper scientific notation.

Answer: 2.5 x 10^1 times (Option B)
