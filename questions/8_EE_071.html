C. 5c + 7
D. 5c + 3
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Cookies: 2c + 5 Brownies: 3c - 2
The problem provides two separate expressions for the amount of chocolate chips used in cookies and brownies.
2
Total = (2c + 5) + (3c - 2) Total = 2c + 3c + 5 - 2
To find the total amount, we need to sum the two given expressions. Remove the parentheses as we are simply adding terms.
3
Combine 2c and 3c: 2c + 3c = 5c Combine 5 and -2: 5 - 2 = 3 Result: 5c + 3
Group terms with the variable c together and constant terms together, then perform the addition/subtraction. The resulting expression 5c + 3 represents the total amount of chocolate chips used.

Answer: 5c + 3 ounces (Option D)

B. (5/4)p + 700
C. (4/6)p + 700
D. (5/4)p + 2300
E. None of the above

Step-by-step solution
1
Decade 1: (1/2)p + 1500 Decade 2: (3/4)p - 800
The problem gives two expressions representing population changes over two decades.
2
Total change = ((1/2)p + 1500) + ((3/4)p - 800) Total change = (1/2)p + (3/4)p + 1500 - 800
To find the total population change, sum the two expressions. Remove the parentheses as it's an addition.
3
Combine (1/2)p and (3/4)p: 1/2 + 3/4 = 2/4 + 3/4 = 5/4 So, (1/2)p + (3/4)p = (5/4)p
Find a common denominator for the fractional coefficients (1/2 and 3/4), which is 4. Convert 1/2 to 2/4, then add the fractions. Combine the constant terms 1500 and -800.
4
Combine 1500 and -800: 1500 - 800 = 700
Subtract the constants. The resulting expression (5/4)p + 700 represents the total population change.

Question 8Grade 8 • EE

B. 5d
C. 5d + 1
D. 5d + 8
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
(1/3)(9d - 12) = (1/3) 9d - (1/3) 12 = 3d - 4
Multiply 1/3 by each term inside the first set of parentheses. 9d/3 = 3d and 12/3 = 4.
2
(1/4)(8d + 16) = (1/4) 8d + (1/4) 16 = 2d + 4
Multiply 1/4 by each term inside the second set of parentheses. 8d/4 = 2d and 16/4 = 4.
3
(3d - 4) + (2d + 4) = 3d + 2d - 4 + 4
Now that both parts of the expression are simplified, add them together. Remove the parentheses.
4
3d + 2d = 5d -4 + 4 = 0 Result: 5d
Group terms with the variable d and constant terms, then perform the addition/subtraction. The resulting expression 5d represents the total distance.

Answer: 5d + 1 miles (Option B)

Question 10Grade 8 • EE
A researcher is studying the growth of a bacterial colony. The growth can be modeled by 1/2(4b + 10) - 1/3(6b - 9) + 5b bacteria, where b is the initial number of bacteria. Which expression is equivalent to the total change in the bacterial colony?
A. 9b + 2
B. 5b + 8
C. 5b + 2
D. 9b + 8
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
1/2(4b + 10) = (1/2) 4b + (1/2) 10 = 2b + 5
Multiply 1/2 by each term inside the first set of parentheses.
2
-1/3(6b - 9) = (-1/3) 6b - (-1/3) 9 = -2b - (-3) = -2b + 3
Multiply -1/3 (including the negative sign) by each term inside the second set of parentheses. Be careful with the double negative.
3
(2b + 5) + (-2b + 3) + 5b = 2b - 2b + 5b + 5 + 3
Now, add all the simplified parts of the expression.
4
2b - 2b + 5b = 5b 5 + 3 = 8 Result: 5b + 8
Group terms with the variable b and constant terms, then perform the addition/subtraction. The resulting expression 5b + 8 represents the total change in the bacterial colony.

Answer: 5b + 8 bacteria (Option B)
