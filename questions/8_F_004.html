Question 10Grade 8 • F
A technician charges a fixed fee of $50 for a service call plus $30 per hour for labor. If the technician works from 1 to 5 hours on a job, what is the range of the possible total charges?
A. {1, 2, 3, 4, 5
B. {$50, $80, $110, $140, $170, $200
C. {$80, $110, $140, $170, $200
D. {$80, $200
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Hours worked (h): {1, 2, 3, 4, 5
The problem states the technician works from 1 to 5 hours. These are the input values.
2
Total Charge (C) = Fixed Fee + (Hourly Rate × Hours Worked) C = $50 + ($30 × h)
The total charge is the sum of the fixed fee and the labor cost, which depends on the hours worked.
3
For h = 1: C = $50 + ($30 × 1) = $80 For h = 2: C = $50 + ($30 × 2) = $110 For h = 3: C = $50 + ($30 × 3) = $140 For h = 4: C = $50 + ($30 × 4) = $170 For h = 5: C = $50 + ($30 × 5) = $200
Substitute each value from the domain into the charge formula to find the corresponding output values.
4
Range: {$80, $110, $140, $170, $200
The set of these calculated total charges represents the range of the function.

Answer: {$80, $110, $140, $170, $200 (Option C)
