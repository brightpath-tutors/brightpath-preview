Question 9Grade 3 • MD
A city planner surveyed residents about their homes' distances from the new park, in miles. The distances were: 1/4, 1/2, 1/4, 3/4, 1, 1/2, 1/4, 1, 3/4, 1/2, 1/4, 1/2. After plotting this data on a line plot, what is the total number of 'X's on the plot?
A. 12
B. 6.5
C. 11
D. 4
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Data: 1/4, 1/2, 1/4, 3/4, 1, 1/2, 1/4, 1, 3/4, 1/2, 1/4, 1/2
List all the given distances from the park.
2
Count for 1/4: four. Count for 1/2: four. Count for 3/4: two. Count for 1: two.
Count the frequency of each unique distance value.
3
Total number of 'X's = 4 (for 1/4) + 4 (for 1/2) + 2 (for 3/4) + 2 (for 1) = 12.
The total number of 'X's on the line plot is the sum of the frequencies of all data points, which is also the total number of data points collected.

Answer: 12 (Option A)

B. 12
C. 4
D. 7
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Data: 1/4, 1/2, 1/4, 3/4, 1, 1/2, 1/4, 1/2, 3/4, 1, 1/4, 1/2
List all the app download times.
2
