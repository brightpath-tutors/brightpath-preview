Question 1Grade 8 • F
The cost of renting a movie is $2 for the first day and $1 for each additional day. Complete the function table to find the total cost of renting a movie for 4 days.
A. $5
B. $3
C. $4
D. $6
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Cost for 1st day = $2. Cost for each additional day = $1.
The problem states the initial cost and the cost for extra days.
2
Total days = 4. First day = 1. Additional days = 4 - 1 = 3 days. Cost = $2 (for 1st day) + $1 * 3 (for additional days) = $2 + $3 = $5.
Add the cost of the first day to the cost of the additional days.

Answer: $5 (Option A)

Question 8Grade 8 • F
The number of active users on a new social media platform doubles every 3 days. If the platform starts with 10 users, complete the function table to find the number of users after 9 days.
A. 90 users
B. 60 users
C. 30 users
D. 80 users
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Initial users = 10. Users double every 3 days.
The problem describes exponential growth. Identify the starting point and the rate of change.
2
Total days = 9. Doubling period = 3 days. Number of periods = 9 / 3 = 3 periods.
Divide the total time by the time it takes for the value to double.
3
Users after 1 period = 10 2 = 20. Users after 2 periods = 20 2 = 40. Users after 3 periods = 40 * 2 = 80.
Multiply the current number of users by 2 for each doubling period.

Answer: 80 users (Option D)
