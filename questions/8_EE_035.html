Question 1Grade 8 • EE
A cashier calculates the total cost of 3 identical items priced at $7 each and 3 identical items priced at $5 each. Using the distributive property, which expression represents the total cost?
A. 3 * (7 + 5)
B. (3 * 7) + 5
C. 3 + 7 + 5
D. 3 7 5
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Cost of first type of item: 3 items $7/item. Cost of second type of item: 3 items $5/item.
We have two groups of items, each with 3 items, but different prices.
2
Total cost = (3 7) + (3 5)
The total cost is the sum of the costs of the two types of items.
3
Since 3 is a common factor in both terms, we can factor it out: 3 * (7 + 5)
The distributive property states that a (b + c) = a b + a * c. We are using it in reverse here to group the common factor.

Answer: 3 * (7 + 5) (Option A)

Question 2Grade 8 • EE
A coach is calculating the total points scored by a team over three games: 8, 5, and 12 points. Which expression shows the total points using the associative property of addition?
A. 8 + 12 + 5
B. 8 * 5 + 12
C. (8 5) 12
D. 8 + (5 + 12)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Scores: 8 points, 5 points, 12 points.
The total points are found by adding these scores together.
2
Total points = 8 + 5 + 12
Without parentheses, addition is performed from left to right.
3
The associative property allows regrouping: (8 + 5) + 12 = 8 + (5 + 12).
This property states that (a + b) + c = a + (b + c), meaning the grouping of numbers in an addition problem does not change the sum.

Answer: 8 + (5 + 12) (Option D)

Question 4Grade 8 • EE
A car travels at a speed of (s + 15) miles per hour for 3 hours. Using the distributive property, write an expression for the total distance traveled.
A. 45s
B. 3s + 45
C. 3s + 15
D. s + 45
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Distance = Speed × Time
This is the fundamental formula for calculating distance.
2
Speed = (s + 15) mph, Time = 3 hours. Distance = 3 * (s + 15)
The speed is a binomial expression, so it must be enclosed in parentheses.
3
3 (s + 15) = (3 s) + (3 * 15) = 3s + 45
The distributive property requires multiplying the factor outside the parentheses by each term inside the parentheses.

Answer: 3s + 45 (Option B)

Question 7Grade 8 • EE
A construction worker is calculating the cost of materials for 1.5 identical units of a project. Each unit requires 8.2 meters of wood at $3.00/meter and 4.1 square meters of tiling at $5.50/square meter. Which expression uses the distributive property to find the total cost of materials?
A. 1.5 (8.2 + 3.00) (4.1 + 5.50)
B. 1.5 (8.2 3.00 + 4.1 * 5.50)
C. 1.5 + (8.2 3.00 + 4.1 5.50)
D. 1.5 8.2 3.00 + 4.1 * 5.50
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Cost of wood = 8.2 meters * $3.00/meter = $24.60
Multiply the length of wood by its price per meter.
2
Cost of tiling = 4.1 sq meters * $5.50/sq meter = $22.55
Multiply the area of tiling by its price per square meter.
3
Cost per unit = (8.2 3.00) + (4.1 5.50)
The total cost for one unit is the sum of the wood and tiling costs.
4
Total cost = 1.5 ( (8.2 3.00) + (4.1 * 5.50) )
Since the cost for each unit is multiplied by the number of units (1.5), the distributive property helps represent this multiplication over the sum of costs per unit.

Answer: 1.5 (8.2 3.00 + 4.1 * 5.50) (Option B)

Question 10Grade 8 • EE
A software update package requires 2(x + 3y) megabytes for core features and 5(2x - y) megabytes for optional features. Which simplified expression represents the total megabytes needed, applying the distributive property?
A. 12x + 11y
B. 4x + 8y
C. 12x - y
D. 12x + y
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
2(x + 3y) = 2 x + 2 3y = 2x + 6y
Multiply the factor 2 by each term inside the first set of parentheses.
2
5(2x - y) = 5 2x - 5 y = 10x - 5y
Multiply the factor 5 by each term inside the second set of parentheses, paying attention to the sign.
3
(2x + 6y) + (10x - 5y)
Add the expressions for core and optional features to get the total.
4
(2x + 10x) + (6y - 5y)
Rearrange terms to put 'x' terms together and 'y' terms together for easier combination.
5
12x + y
Add the coefficients of the 'x' terms and the coefficients of the 'y' terms.

Answer: 12x + y (Option D)
