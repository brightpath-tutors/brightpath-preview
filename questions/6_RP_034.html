Question 1Grade 6 • RP
A cyclist's distance traveled over time is shown on the graph below. Does the graph represent a proportional relationship?
A. Yes, because the distance is always increasing.
B. Yes, because the graph is a straight line that passes through the origin.
C. No, because the line is not perfectly vertical.
D. No, because the distance increases with time.
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
The graph forms a straight line.
A key characteristic of a proportional relationship is that its graph must be a straight line.
2
The line passes through the point (0,0).
For a relationship to be proportional, when one quantity is zero, the other quantity must also be zero. This means the graph must pass through the origin.
3
Since the graph is a straight line and passes through the origin, it represents a proportional relationship.
Both conditions (straight line and passing through the origin) must be met for a relationship to be proportional.

Answer: Yes, because the graph is a straight line that passes through the origin. (Option B)

Question 2Grade 6 • RP
A painter uses a certain amount of paint for each canvas. The graph shows the amount of paint used (in ounces) for different numbers of canvases. Based on the graph, how many ounces of paint does the painter use for each canvas?
A. 1 ounce
B. 9 ounces
C. 6 ounces
D. 3 ounces
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Choose points like (1, 3) or (2, 6).
To find the unit rate, we can use any point (x, y) on the line where x is not zero, and calculate y/x.
2
Using (1, 3): 3 ounces / 1 canvas = 3 ounces per canvas. Using (2, 6): 6 ounces / 2 canvases = 3 ounces per canvas.
The unit rate is the amount of paint used per single canvas, which corresponds to the value of y when x is 1, or the ratio y/x.

Answer: 3 ounces per canvas. (Option D)

Question 3Grade 6 • RP
A student tracks the number of practice problems solved over several days. The graph shows this relationship. How many practice problems does the student solve per day?
A. 20 problems
B. 2 problems
C. 10 problems
D. 5 problems
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let's use the point (2, 10).
Any point (x, y) on a proportional graph (except the origin) can be used to find the unit rate.
2
Unit rate = Number of problems / Number of days = 10 problems / 2 days = 5 problems per day.
The unit rate represents the amount of the dependent variable (problems) per unit of the independent variable (days).

Answer: 5 problems per day. (Option D)

The unit rate can be directly read from the graph where the x-value is 1, or calculated as y/x for any point.

Question 6Grade 6 • RP
A scientist measures the mass of a growing crystal over several days. The graph shows the crystal's mass in grams (g) after a certain number of days. Which equation represents the relationship between the crystal's mass (y) and time in days (x)?
A. y = x + 0.4
B. y = 1.2x
C. y = 3x
D. y = 0.4x
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
The graph is a straight line passing through the origin (0,0), so it represents a proportional relationship.
This means the relationship can be written in the form y = kx, where k is the constant of proportionality (unit rate).
2
Using point (3, 1.2): k = y/x = 1.2 g / 3 days = 0.4 g/day.
The constant of proportionality is the unit rate, representing the mass gained per day.
3
Substitute k = 0.4 into y = kx, which gives y = 0.4x.
This equation allows us to find the mass (y) for any given number of days (x).

Answer: y = 0.4x (Option D)

Question 7Grade 6 • RP
A bricklayer lays bricks at a steady rate. The graph shows the number of bricks laid versus the time in hours. If the bricklayer needs to lay 1000 bricks, how many hours will it take?
A. 10 hours
B. 8 hours
C. 7.5 hours
D. 6 hours
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
From the point (2, 250), unit rate = 250 bricks / 2 hours = 125 bricks per hour.
This is the constant of proportionality, k. So, y = 125x.
2
We want to find x when y = 1000. So, 1000 = 125x. x = 1000 / 125 = 8 hours.
Divide the total number of bricks by the rate per hour to find the total hours.

Answer: 8 hours. (Option B)

Question 10Grade 6 • RP
A data server transfers files. The graph below shows the amount of data transferred in megabytes (MB) over different durations of time in seconds. Which statement accurately describes the data transfer and provides a correct justification?
A. The data transfer rate is constant at 4 MB/second because the graph is a straight line passing through the origin.
B. The data transfer is not proportional because it starts at 0.
C. The data transfer rate is increasing because the line goes upwards.
D. The data transfer rate is 20 MB/second because 20 MB is transferred in 5 seconds.
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
The graph is a straight line passing through the origin (0,0).
This means the amount of data transferred is directly proportional to the time taken.
2
Using point (5, 20), k = y/x = 20 MB / 5 seconds = 4 MB/second.
The constant of proportionality represents the data transfer rate.
3
The relationship is proportional, and the unit rate is 4 MB/second.
The correct statement must reflect both the proportionality and the correct unit rate.

Answer: The data transfer rate is constant at 4 MB/second because the graph is a straight line passing through the origin. (Option A)
