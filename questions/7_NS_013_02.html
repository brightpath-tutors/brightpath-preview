Question 1Grade 7 • NS
A shop tracks the daily change in the value of an antique vase. For the past 3 days, the value has consistently decreased by $1.25 per day. If this trend is represented by a daily change of -$1.25, what is the total change in the vase's value over these 3 days?
A. $3.75
B. -$37.50
C. $1.75
D. -$3.75
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Number of days = 3 Daily change in value = -$1.25
Identify the given quantities in the problem.
2
Operation: 3 * (-$1.25)
To find the total change, multiply the number of days by the daily change in value.
3
Absolute value multiplication: 3 * 1.25 = 3.75
Multiply the absolute values of the numbers first.
4
Sign of the product: Positive * Negative = Negative
Apply the rule for multiplying integers: a positive number multiplied by a negative number results in a negative number.
5
Total change in value = -$3.75
Combine the absolute value and the sign to get the final answer.

Answer: -$3.75 (Option D)
