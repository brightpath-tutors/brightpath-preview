Question 1Grade 9 • A-CED
A mobile game charges a fixed subscription fee of $5 and an additional $2 for each premium item purchased. Write an algebraic expression for the total cost, C, if p premium items are purchased.
A. C = 5p + 2
B. C = 7p
C. C = 2p - 5
D. C = 5 + 2p
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Fixed cost = $5
Identify the one-time, fixed cost.
2
Cost per premium item = $2
Identify the cost associated with each variable item.
3
Number of premium items = p
Identify the variable representing the number of items purchased.
4
Variable cost = 2 * p = 2p
Multiply the cost per item by the number of items to find the total variable cost.
5
Total cost C = Fixed cost + Variable cost = 5 + 2p
Add the fixed cost to the total variable cost to get the algebraic expression for the total cost.

Answer: C = 5 + 2p (Option D)

Question 8Grade 9 • A-CED

A. R = 200 + 75t
B. R = 200t + 75
C. R = 200 + 75(t - 3)
D. R = 75t - 200
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Base fee = $200
Identify the initial fixed cost for the first block of time.
2
Base hours covered by fee = 3 hours
Identify the duration covered by the base fee.
3
Additional hourly rate = $75
Identify the rate for hours exceeding the base duration.
4
Total rental hours = t (where t > 3)
Identify the total hours, which is the variable.
5
Hours subject to additional charge = t - 3
Subtract the base hours from the total hours to find how many hours are charged at the additional rate.
6
Additional cost = 75 * (t - 3) = 75(t - 3)
Multiply the additional hourly rate by the number of hours subject to that charge.
7
Total rental cost R = Base fee + Additional cost = 200 + 75(t - 3)
Add the base fee to the additional cost to get the algebraic expression for the total rental cost.

Answer: R = 200 + 75(t - 3) (Option C)
