Question 1Grade 7 • EE
Sarah bought a book for $12.75 and a magazine. She spent a total of $18.50. How much did the magazine cost? Solve this problem using an algebraic approach by setting up and solving an equation.
A. $31.25
B. $6.75
C. $5.75
D. $1.45
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let m = cost of the magazine. 12.75 + m = 18.50
Define a variable for the unknown and set up an equation representing the total cost.
2
m = 18.50 - 12.75
To isolate 'm', subtract the cost of the book from the total amount spent on both items.
3
m = 5.75
Perform the subtraction to find the cost of the magazine.

Answer: $5.75 (Option C)

Question 4Grade 7 • EE
Emily drove for 4 hours at a constant speed. If she had driven 15 mph faster, she would have covered 340 miles. What was her actual average speed in miles per hour? Solve this problem using an algebraic approach by setting up and solving an equation.
A. 85 mph
B. 70 mph
C. 65 mph
D. 90 mph
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let s = Emily's actual average speed in mph. 4(s + 15) = 340
Define a variable for her actual speed, then represent the hypothetical speed as (s + 15). The equation is (time) * (speed) = (distance).
2
4s + 60 = 340
Apply the distributive property to multiply 4 by each term inside the parentheses.
3
4s = 340 - 60
Subtract 60 from both sides of the equation to start isolating 's'.
4
4s = 280
Perform the subtraction.
5
s = 280 / 4
Divide both sides by 4 to solve for 's'.
6
s = 70
Perform the division to find Emily's actual average speed.

Answer: 70 mph (Option B)

A. 215 minutes
B. 205 minutes
C. 225 minutes
D. 195 minutes
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Total duration = 1:00 PM - 8:30 AM = 4 hours 30 minutes = 270 minutes. Let t = actual instructional time in minutes.
Calculate the total duration of the workshop in minutes. Define a variable for the actual instructional time.
2
t + 45 + 2(10) = 270
Set up an equation where instructional time plus lunch break time plus coffee break times equals the total duration.
3
t + 45 + 20 = 270
Calculate the total time for the coffee breaks.
4
t + 65 = 270
Combine all known non-instructional times (lunch and coffee breaks).
5
t = 270 - 65
Subtract the total non-instructional time from the total workshop duration to isolate 't'.
6
t = 205
Perform the subtraction to find the actual instructional time.

Answer: 205 minutes (Option B)

Question 9Grade 7 • EE
The population of a small town decreased by 20% last year due to economic reasons. If the current population is 12,400, what was the population last year? Solve this problem using an algebraic approach by setting up and solving an equation.
A. 14,880
B. 10,333
C. 15,500
D. 14,500
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let P = population last year. P - 0.20P = 12400
Define a variable for last year's population. A 20% decrease means 20% of P is subtracted from P. 20% as a decimal is 0.20.
2
1P - 0.20P = 12400
Rewrite P as 1P for clarity in subtraction.
3
0.80P = 12400
Combine like terms (1P - 0.20P).
4
P = 12400 / 0.80
Divide both sides by 0.80 to solve for P.
5
P = 15500
Perform the division to find the population last year.

Answer: 15,500 (Option C)

Question 10Grade 7 • EE
A cell phone plan costs $45 per month plus $0.08 per MB of data used. If a user's bill last month was $69.96, how many MB of data did they use? Solve this problem using an algebraic approach by setting up and solving an equation.
A. 312 MB
B. 874.5 MB
C. 114.96 MB
D. 69.96 MB
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let d = number of MB of data used. 45 + 0.08d = 69.96
Define a variable for the unknown data usage. Set up an equation: fixed cost + (cost per MB * MB used) = total bill.
2
0.08d = 69.96 - 45
Subtract the fixed monthly cost from the total bill to find the cost attributable to data usage.
3
0.08d = 24.96
Perform the subtraction.
4
d = 24.96 / 0.08
Divide the data cost by the cost per MB to find the total MB used.
5
d = 312
Perform the division to find the number of MB of data used.

Answer: 312 MB (Option A)
