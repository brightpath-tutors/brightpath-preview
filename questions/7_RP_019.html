Question 1Grade 7 • RP
A toy car in a store flyer is 4 cm long. The actual toy car is 20 cm long. If the toy car in the flyer is 2 cm tall, what is the actual height of the toy car?
A. 40 cm
B. 8 cm
C. 18 cm
D. 10 cm
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Given: Flyer Length = 4 cm, Actual Length = 20 cm, Flyer Height = 2 cm. Find Actual Height (x).
Identify the given dimensions and the unknown value. Recognize that similar figures imply proportional relationships between corresponding sides.
2
Flyer Length / Actual Length = Flyer Height / Actual Height 4 cm / 20 cm = 2 cm / x cm
Form a proportion by comparing the ratio of corresponding lengths (flyer to actual) for both dimensions (length and height).
3
4 x = 20 2 4x = 40 x = 40 / 4 x = 10
Use cross-multiplication to solve the proportion. Multiply the numerator of the first ratio by the denominator of the second, and vice-versa. Then isolate x.
4
The actual height of the toy car is 10 cm.
The value of x represents the actual height of the toy car.

Answer: 10 cm (Option D)

Question 9Grade 7 • RP
A miniature model of a city has a scale of 1:1000. If a park in the model is 5 cm wide and 8 cm long, what is the actual perimeter of the park in meters?
A. 26 cm
B. 26000 meters
C. 130 meters
D. 260 meters
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Given: Scale = 1:1000, Model width = 5 cm, Model length = 8 cm. Find Actual perimeter in meters.
Interpret the given scale factor to understand the relationship between model dimensions and actual dimensions.
2
Actual width = 5 cm * 1000 = 5000 cm Actual length = 8 cm * 1000 = 8000 cm
Multiply each model dimension by the scale factor (1000) to get the actual dimensions in centimeters.
3
Actual width = 5000 cm / 100 cm/meter = 50 meters Actual length = 8000 cm / 100 cm/meter = 80 meters
Divide the actual dimensions in centimeters by 100 to convert them into meters.
4
Perimeter = 2 * (width + length) Perimeter = 2 * (50 meters + 80 meters) Perimeter = 2 * (130 meters) Perimeter = 260 meters
Use the formula for the perimeter of a rectangle (2 * (length + width)) with the actual dimensions in meters.
5
The actual perimeter of the park is 260 meters.
The calculated value is the actual perimeter in the requested unit.

Answer: 260 meters (Option D)

Question 10Grade 7 • RP
A company is designing a new smartphone. The prototype screen has a width of 7.2 cm and a height of 12.8 cm. The final production model will have a similar screen, but its width will be 6.3 cm. What will be the area of the final production model's screen, in square centimeters?
A. 70.56 sq cm
B. 92.16 sq cm
C. 17.5 sq cm
D. 80.64 sq cm
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Given: Prototype Width = 7.2 cm, Prototype Height = 12.8 cm, New Width = 6.3 cm. Find New Area (A_new).
Identify the given dimensions for the prototype and the new model. Recognize that similar figures imply a consistent linear scale factor and a squared area scale factor.
2
k = New Width / Prototype Width k = 6.3 cm / 7.2 cm k = 0.875
Calculate the ratio of the new width to the original width to find the linear scale factor.
3
New Height = Prototype Height * k New Height = 12.8 cm * 0.875 New Height = 11.2 cm
Multiply the prototype's height by the linear scale factor to find the new height.
4
Area = New Width * New Height Area = 6.3 cm * 11.2 cm Area = 70.56 sq cm
Multiply the new width by the new height to find the area of the final screen.
5
Area_prototype = 7.2 cm * 12.8 cm = 92.16 sq cm Area_new = Area_prototype * k² Area_new = 92.16 sq cm * (0.875)² Area_new = 92.16 sq cm * 0.765625 Area_new = 70.56 sq cm
Calculate the prototype area, then multiply by the square of the linear scale factor to find the new area. This is a more advanced but efficient method for similar figures.
6
The area of the final production model's screen will be 70.56 sq cm.
The calculated value is the area in the requested unit.

Answer: 70.56 sq cm (Option A)
