Question 1Grade 9 • A-CED
A bakery sells cupcakes for $3.50 each. They want to make at least $250 in cupcake sales today. Write an inequality to represent the number of cupcakes (c) the bakery needs to sell.
A. 3.50 + c >= 250
B. 3.50c > 250
C. 3.50c >= 250
D. c / 3.50 >= 250
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Cost per cupcake = $3.50; Number of cupcakes = c
Identify the cost per cupcake and the variable representing the number of cupcakes.
2
Total sales = 3.50 * c
Formulate the algebraic expression for the total sales by multiplying the cost per cupcake by the number of cupcakes.
3
Condition: Total sales must be 'at least' $250. 'At least' translates to >=.
Interpret the phrase 'at least $250' as greater than or equal to.
4
3.50c >= 250
Combine the total sales expression and the inequality condition to form the final inequality.

Answer: 3.50c >= 250 (Option C)

B. 3x + 2y = 25

🔒 Backend review only — not visible to students
Step-by-step solution

Regular basket points = 2; Three-pointer points = 3

B. 4s + 3(s + 20) < 400
C. 4s + 3s + 20 < 400
D. 4(s + 3) + 3(s + 20) < 400
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Segment 1: Speed = s, Time = 4 hours; Segment 2: Speed = s + 20, Time = 3 hours; Total distance limit = 400 km
Identify variables and given values for speed, time, and the total distance limit.
2
Distance for segment 1 = speed * time = 4s
Calculate the distance covered in the first segment using the formula distance = speed × time.
3
Distance for segment 2 = speed * time = 3(s + 20)
Calculate the distance covered in the second segment, remembering to use parentheses for the speed expression.
4
Total distance = 4s + 3(s + 20)
Add the distances from both segments to find the total distance traveled.
5
Condition: Total distance is 'less than' 400 km. 'Less than' translates to <.
Interpret the phrase 'less than 400 kilometers' as a strict inequality.
6
4s + 3(s + 20) < 400
Combine the total distance expression and the inequality condition to form the final inequality.

Answer: 4s + 3(s + 20) < 400 (Option B)

Question 8Grade 9 • A-CED

B. 2.5h >= 50 and 2.5h <= 75
C. 2.5h = 50 and 2.5h = 75
D. 2.5h > 50 and 2.5h < 75
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Weekday hours = h; Weekend hours = 1.5h
Define the variable for weekday hours and express weekend hours based on it.
2
Total hours = h + 1.5h = 2.5h
Calculate the total hours worked by summing weekday and weekend hours.
3
Constraint 1: Minimum of 50 hours. Total hours >= 50 => 2.5h >= 50.
Translate the 'minimum hours' constraint into an inequality using the total hours expression.
4
Constraint 2: Cannot work more than 75 hours. Total hours <= 75 => 2.5h <= 75.

B. 1200 < d + r < 2000 and d <= (1/3)r

E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Number of deer = d; Number of rabbits = r
Identify the variables representing the populations of deer and rabbits.
2
Constraint 1: Total animals between 1200 and 2000, inclusive. This means 1200 <= d + r <= 2000.

3

The battery life (b, in hours) must be directly proportional to the screen size, with a constant of proportionality of 4.2, and the battery life must be at least 10 hours. B. 1.5 < s < 2.1, b = 4.2s, b >= 10

🔒 Backend review only — not visible to students
Step-by-step solution
1
Screen size = s (inches); Battery life = b (hours)
Identify the variables for screen size and battery life.
2
Constraint 1: Screen size 'at least 1.5 inches but no more than 2.1 inches'. This means 1.5 <= s <= 2.1.
Translate the range constraint for the screen size, noting 'at least' and 'no more than' imply inclusive bounds.
3
Constraint 2: Battery life (b) is 'directly proportional' to screen size (s) with constant 4.2. This means b = 4.2s.
