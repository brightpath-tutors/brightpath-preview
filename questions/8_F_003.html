B. 8 dollars
C. 10 dollars
D. 11 dollars
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
The function C(x) = 2x + 1 gives the total cost for buying x notebooks. We need to find the cost for 5 notebooks, so we need to evaluate C(5).
Identify what the function represents and what value of x needs to be used.
2
Replace x with 5 in the function: C(5) = 2(5) + 1
Substitute the given number of notebooks into the function expression.
3
According to the order of operations, perform multiplication first: 2 * 5 = 10. So, C(5) = 10 + 1
Multiply the coefficient by the substituted value of x.
4
Finally, perform the addition: 10 + 1 = 11. So, C(5) = 11
Add the remaining terms to find the final cost.

Answer: 11 dollars (Option D)
