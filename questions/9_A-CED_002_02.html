Question 1Grade 9 • A-CED
The height of a ball thrown upwards is given by the expression \(h(t) = 20t - 5t^2\), where \(t\) is the time in seconds after the ball is thrown. What is the algebraic expression for the height of the ball if it is thrown from a platform 3 meters high?
A. \(h(t) = 20t - 5t^2 - 3\)
B. \(h(t) = 23t - 5t^2\)
C. \(h(t) = -5t^2 + 20t + 3\)
D. \(h(t) = 20t + 5t^2 + 3\)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original height expression: \(h(t) = 20t - 5t^2\)
Identify the given algebraic expression for the height of the ball.
2
Additional height from platform = 3 meters
Identify the additional height provided by the platform.
3
\(h(t) = (20t - 5t^2) + 3\)
Add the initial height from the platform to the original expression, as it represents a constant vertical shift.
4
\(h(t) = -5t^2 + 20t + 3\)
Rewrite the expression in standard quadratic form (\(at^2 + bt + c\)). The height is a quadratic relationship with respect to time, and the constant term 'c' represents the initial height or starting point.

Answer: \(h(t) = -5t^2 + 20t + 3\) (Option C)

Question 2Grade 9 • A-CED
A company's daily profit \(P\) (in hundreds of dollars) is modeled by the equation \(P = -x^2 + 10x\), where \(x\) is the number of items sold (in hundreds). If the company introduces a new product that increases daily overhead by $7 hundred, how does the profit equation change?
A. \(P = -x^2 + 10x + 7\)
B. \(P = -x^2 + 17x\)
C. \(P = -x^2 + 10x - 7\)
D. \(P = -x^2 + 3x\)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original profit equation: \(P = -x^2 + 10x\)
Identify the given algebraic expression for the company's daily profit.
2
Increase in daily overhead = $7 hundred
Identify the change in overhead cost.
3
\(P = (-x^2 + 10x) - 7\)
Since overhead is a cost, an increase in overhead reduces profit. Therefore, subtract the increase in overhead from the original profit equation.
4
\(P = -x^2 + 10x - 7\)
State the final profit equation.

Answer: \(P = -x^2 + 10x - 7\) (Option C)

Question 3Grade 9 • A-CED
The path of a toy rocket is modeled by \(h(t) = -2t^2 + 15t\), where \(h(t)\) is the height in meters and \(t\) is the time in seconds. If the rocket's launch velocity increases, adding $4t$ to its height function, what is the new expression for the rocket's height?
A. \(h(t) = -2t^2 + 15t - 4\)
B. \(h(t) = -2t^2 + 60t\)
C. \(h(t) = -2t^2 + 19t\)
D. \(h(t) = -2t^2 + 4t - 15\)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original height expression: \(h(t) = -2t^2 + 15t\)
Identify the initial height function of the toy rocket.
2
Additional height component due to increased velocity: $4t$
Identify the term that needs to be added to the function.
3
\(h(t) = (-2t^2 + 15t) + 4t\)
Add the additional height component to the original expression.
4
\(h(t) = -2t^2 + (15t + 4t)\)
Group like terms for simplification.
5
\(h(t) = -2t^2 + 19t\)
Combine the 't' terms to get the new height function. This represents a quadratic relationship where the coefficient of the 't' term (linear term) has changed due to increased velocity.

If the cost of ingredients increases, reducing the revenue by $2 for every dollar increase in price, and a new fixed cost of $25 is introduced, what is the new revenue function?
A. \(R(p) = -10p^2 + 152p + 25\)
B. \(R(p) = -12p^2 + 150p - 25\)
C. \(R(p) = -10p^2 + 148p - 25\)

🔒 Backend review only — not visible to students
Step-by-step solution
1
Original revenue function: \(R(p) = -10p^2 + 150p\)
Identify the initial revenue function.
2
Reduction in revenue due to ingredient cost increase: $2p$
An increase in ingredient cost reduces revenue by $2 for every dollar increase in price, so this is represented by $-2p$.
3
New fixed cost: $25
A fixed cost reduces the overall revenue, so this is represented by $-25$.
4
\(R(p) = (-10p^2 + 150p) - 2p - 25\)
Combine the original revenue function with the reductions in revenue.
5
\(R(p) = -10p^2 + (150p - 2p) - 25\)
Group like terms for simplification.
6
\(R(p) = -10p^2 + 148p - 25\)

Question 5Grade 9 • A-CED
A new cooking technique changes the temperature profile in an oven. The original temperature \(T\) (in degrees Celsius) at time \(t\) (in minutes) is given by \(T(t) = -0.5t^2 + 15t + 100\). The new technique uniformly increases the temperature by $10$ degrees and also introduces a cooling effect proportional to the square of the time, represented by $-0.1t^2$. What is the new temperature function?
A. \(T(t) = -0.4t^2 + 15t + 110\)
B. \(T(t) = -0.6t^2 + 15t + 90\)
C. \(T(t) = -0.6t^2 + 15t + 110\)
D. \(T(t) = -0.5t^2 + 14.9t + 110\)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original temperature function: \(T(t) = -0.5t^2 + 15t + 100\)
Identify the initial temperature function.
2
Uniform temperature increase: $10$ degrees
This is a constant term added to the function.
3
Cooling effect proportional to square of time: $-0.1t^2$
This is a quadratic term added to the function.
4
\(T(t) = (-0.5t^2 + 15t + 100) + 10 - 0.1t^2\)
Combine all components to form the new temperature function.
5
\(T(t) = (-0.5t^2 - 0.1t^2) + 15t + (100 + 10)\)
Group like terms for simplification.
6
\(T(t) = -0.6t^2 + 15t + 110\)
Combine the \(t^2\) terms and the constant terms to get the final temperature function. This maintains the quadratic nature of the relationship, with changes to the quadratic and constant coefficients.

Answer: \(T(t) = -0.6t^2 + 15t + 110\) (Option C)

Question 6Grade 9 • A-CED

If a new school policy encourages participation, adding $5x$ students but also causing a fixed group of $8$ students to leave, how does the expression for \(N(x)\) change?
A. \(N(x) = -2x^2 + 20x + 58\)
B. \(N(x) = -2x^2 + 30x + 42\)
C. \(N(x) = -2x^2 + 30x + 58\)
D. \(N(x) = -2x^2 + 20x + 42\)
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original expression: \(N(x) = -2x^2 + 25x + 50\)
Identify the initial expression for student participation.
2
Increase in participation due to new policy: $5x$
This term represents an addition to the number of students.
3
Fixed group of students leaving: $8$
This is a constant term that needs to be subtracted from the total.

5
\(N(x) = -2x^2 + (25x + 5x) + (50 - 8)\)
Group like terms for simplification.
6
\(N(x) = -2x^2 + 30x + 42\)

1
Original growth function: \(G(d) = 0.8d^2 - 3d + 100\)
Identify the initial function for bacterial growth.
2
Boost in initial colony size: $15$ thousand

3
Accelerated growth: $0.2d^2$ thousand bacteria per day

Original trajectory equation: \(y = -0.05x^2 + 2.5x + 1.2\)

2
Increase in initial height from platform: $0.8$ meters
This is a constant term added to the function.

Additional height due to adjusted launch angle: $0.02x^2$

Due to unexpected environmental factors, the integrity is reduced by $0.03t^2$ and an initial flaw causes a further reduction of $7.5$ units. What is the new expression for the bridge's structural integrity?
A. \(S(t) = -0.09t^2 + 5.5t + 107.5\)
B. \(S(t) = -0.15t^2 + 5.5t + 92.5\)
C. \(S(t) = -0.12t^2 + 5.5t + 92.5 - 0.03t^2\)

E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Original structural integrity function: \(S(t) = -0.12t^2 + 5.5t + 100\)
Identify the initial function for the bridge's structural integrity.
2
Reduction due to environmental factors: $-0.03t^2$
This is a quadratic term representing a decrease in integrity over time.
3
Reduction due to initial flaw: $-7.5$ units

B. \(P(y) = 0.015y^2 - 0.5y + 105\)
C. \(P(y) = 0.01y^2 - 0.5y + 135 + 0.005y^2\)

🔒 Backend review only — not visible to students
Step-by-step solution
1
Original population function: \(P(y) = 0.01y^2 - 0.5y + 120\)
Identify the initial population function.
2
Constant population increase: $15$ thousand people
This is a constant term added to the function.
3
Additional population growth rate: $0.005y^2$ thousand people per year

\(P(y) = (0.01y^2 - 0.5y + 120) + 15 + 0.005y^2\)
