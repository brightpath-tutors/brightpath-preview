Question 6Grade 7 • NS
A scientist measures the area of a circular bacteria colony to be 25π square millimeters. Which type of number best describes this measurement?
A. Whole Number
B. Irrational Number
C. Integer
D. Rational Number
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
A rational number can be written as a fraction p/q (integers, q≠0). An irrational number cannot be written in this form; its decimal representation is non-terminating and non-repeating.
These are the fundamental definitions for classifying real numbers.
2
The measurement is 25π. This is a product of 25 and π.
We need to determine the type of each component.
3
- 25: This is an integer, and therefore a rational number (25/1). - π (pi): This is a known irrational number, with a non-repeating, non-terminating decimal expansion (e.g., 3.14159...).
Recognizing π as irrational is key.
4
The product of a non-zero rational number (25) and an irrational number (π) is always an irrational number.
This is a property of real numbers. So, 25π is an irrational number.

Answer: Irrational Number (Option B)

E. None of the above

Step-by-step solution
1
The length of the beam is √72 feet.
This involves a square root.
2
√72 = √(36 2) = √36 √2 = 6√2.
We look for perfect square factors to simplify the radical.
3

3
1.2 x 10^-4 = 0.00012
Moving the decimal point 4 places to the left for 10^-4.
4
0.00012 is a terminating decimal. Any terminating decimal can be written as a fraction of two integers (e.g., 12/100000).
