Question 1Grade 8 • F
A store's daily profit (in hundreds of dollars) can be modeled by the quadratic function P(x) = -x^2 + 8x - 12, where x is the number of hours the store is open. What is the vertex of the parabola represented by this function, and does it open upwards or downwards?
A. Vertex (4, 4), opens upwards
B. Vertex (4, 4), opens downwards
C. Vertex (4, -4), opens downwards
D. Vertex (-4, 4), opens downwards
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For P(x) = -x^2 + 8x - 12, a = -1, b = 8, c = -12.
The coefficients a, b, and c are identified from the standard quadratic form P(x) = ax^2 + bx + c.
2
Since a = -1 (which is negative), the parabola opens downwards.
If the leading coefficient 'a' is negative, the parabola opens downwards. If 'a' is positive, it opens upwards.
3
The x-coordinate of the vertex is given by the formula x = -b / (2a). x = -8 / (2 * -1) = -8 / -2 = 4.
This formula provides the x-coordinate (or axis of symmetry) of the parabola's vertex.
4
Substitute the x-coordinate (x = 4) back into the function: P(4) = -(4)^2 + 8(4) - 12 = -16 + 32 - 12 = 16 - 12 = 4.
Substituting the x-coordinate into the original function gives the corresponding y-coordinate of the vertex.
5
The vertex is (4, 4) and the parabola opens downwards.
Combine the calculated vertex coordinates with the determined direction of opening.

Answer: Vertex (4, 4), opens downwards. (Option B)

State the complete description of the graph based on the function type and leading coefficient.

The number of students attending a school concert over several years can be modeled by the function N(y) = -0.1y^2 + 2y + 100, where N(y) is the number of students and y is the number of years since the school opened. What is the axis of symmetry for the graph of this function?
A. y = 10
B. y = -10
C. x = 10
D. y = 2
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For N(y) = -0.1y^2 + 2y + 100, a = -0.1, b = 2, c = 100.
Extract the values for a, b, and c from the quadratic function.
2
The axis of symmetry for a quadratic function in the form f(x) = ax^2 + bx + c is given by x = -b / (2a). In this case, the independent variable is 'y', so the formula is y = -b / (2a). y = -2 / (2 * -0.1) = -2 / -0.2 = 10.
The axis of symmetry is a vertical line that passes through the vertex of the parabola. Its equation is derived from the quadratic formula.
3
The axis of symmetry is y = 10.

A. Vertex (10, 35). This means the maximum concentration of the chemical is 35 mg/L after 10 hours.
B. Vertex (5, 10). This means the concentration is 10 mg/L after 5 hours.
C. Vertex (-10, 35). This means the maximum concentration of the chemical is 35 mg/L after -10 hours.
D. Vertex (10, 35). This means the minimum concentration of the chemical is 35 mg/L after 10 hours.
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For C(t) = -0.25t^2 + 5t + 10, a = -0.25, b = 5, c = 10.
Extract the coefficients from the given quadratic function.
2
t = -b / (2a) = -5 / (2 * -0.25) = -5 / -0.5 = 10.
The t-coordinate of the vertex represents the time at which the maximum or minimum concentration occurs.
3
Substitute t = 10 into the function: C(10) = -0.25(10)^2 + 5(10) + 10 C(10) = -0.25(100) + 50 + 10 C(10) = -25 + 50 + 10 C(10) = 25 + 10 = 35.
This calculation gives the actual concentration at the time found in the previous step.
4
The vertex is (10, 35). Since a = -0.25 (negative), the parabola opens downwards, so the vertex is a maximum point. This means the maximum concentration of the chemical is 35 mg/L, occurring after 10 hours.
A negative 'a' value indicates a maximum point, while a positive 'a' value indicates a minimum point. The coordinates of the vertex represent the optimal time and corresponding concentration.

A. Parabola opening downwards, with incorrect vertex (e.g., (7, -25)).
B. Parabola opening downwards, with incorrect x-intercepts (e.g., (4,0) and (6,0)).
C. Parabola opening downwards, with vertex (7, 25) and x-intercepts (2,0) and (12,0).
D. Parabola opening upwards.
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For V(d) = -d^2 + 14d - 24, a = -1, b = 14, c = -24. Since a = -1 (negative), the parabola opens downwards.
The sign of 'a' determines if the parabola opens upwards or downwards.
2
d-coordinate of vertex: d = -b / (2a) = -14 / (2 * -1) = -14 / -2 = 7. V(7) = -(7)^2 + 14(7) - 24 = -49 + 98 - 24 = 49 - 24 = 25. Vertex is (7, 25).
The vertex is a key feature that indicates the maximum or minimum point of the parabola.
3
-d^2 + 14d - 24 = 0 d^2 - 14d + 24 = 0 (multiply by -1) Factor the quadratic: (d - 2)(d - 12) = 0 So, d = 2 or d = 12. The x-intercepts are (2,0) and (12,0).
The x-intercepts are the points where the graph crosses the horizontal axis (V(d)=0). Factoring is one method to find these.

The y-intercept indicates the value of V(d) when d is zero.
5
The graph must be a parabola opening downwards, with its vertex at (7, 25) and crossing the x-axis at (2,0) and (12,0), and the y-axis at (0, -24).
Combine all determined features to correctly identify the graph. Knowing the vertex and x-intercepts are often sufficient to distinguish graphs.

Answer: A parabola opening downwards, with vertex (7, 25) and x-intercepts (2,0) and (12,0). (Option C)

A. Parabola opening upwards, with vertex (20, 13).
B. Parabola opening downwards, with incorrect vertex (e.g., (40, 5)).

🔒 Backend review only — not visible to students
Step-by-step solution
1
For F(u) = -0.02u^2 + 0.8u + 5, a = -0.02, b = 0.8, c = 5. Since a = -0.02 (negative), the parabola opens downwards.
The sign of 'a' determines the direction of the parabola. Negative 'a' means it opens downwards.
2
