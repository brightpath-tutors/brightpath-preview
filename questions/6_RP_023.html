Question 1Grade 6 • RP
If 3 pencils cost $0.75, how much do 5 pencils cost?
A. $1.25
B. $0.45
C. $2.25
D. $0.15
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let \(x\) be the cost of 5 pencils. $\frac{\text{Number of pencils}}{\text{Cost}} = \frac{3}{0.75} = \frac{5}{x
We set up a proportion comparing the ratio of pencils to cost for both scenarios. Make sure the units are consistent across both ratios.
2
$3 \times x = 0.75 \times 5$ $3x = 3.75$
To solve a proportion, we cross-multiply the terms. This means multiplying the numerator of one ratio by the denominator of the other and setting them equal.
3
\(x = \frac{3.75}{3}\) \(x = 1.25\)
Divide both sides of the equation by 3 to find the value of \(x\), which represents the cost of 5 pencils.

Answer: $1.25 (Option A)

Question 2Grade 6 • RP
A baseball player hit 4 home runs in 16 games. At this rate, how many home runs would he hit in 28 games?
A. 64 home runs
B. 32 home runs
C. 7 home runs
D. 16 home runs
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let \(x\) be the number of home runs in 28 games. $\frac{\text{Home runs}}{\text{Games}} = \frac{4}{16} = \frac{x}{28
We set up a proportion comparing the ratio of home runs to games for both situations. Ensure consistent units.
2
$4 \times 28 = 16 \times x$ $112 = 16x$
Cross-multiply the terms to form an equation.
3
\(x = \frac{112}{16}\) \(x = 7\)
Divide both sides by 16 to find the value of \(x\), which is the number of home runs.

Answer: 7 home runs (Option C)

Question 4Grade 6 • RP
A car travels 150 miles in 3 hours. How long will it take to travel 225 miles at the same speed?
A. 4.5 hours
B. 78 hours
C. 1.5 hours
D. 675 hours
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let \(x\) be the time it takes to travel 225 miles. $\frac{\text{Distance (miles)}}{\text{Time (hours)}} = \frac{150}{3} = \frac{225}{x
We set up a proportion comparing distance to time. This allows us to calculate the time for a different distance at a constant speed.
2
$150 \times x = 3 \times 225$ $150x = 675$
Cross-multiply the values to form an equation that can be solved for \(x\).
3
\(x = \frac{675}{150}\) \(x = 4.5\)
Divide both sides by 150 to find the unknown time in hours.

Answer: 4.5 hours (Option A)

Question 8Grade 6 • RP
A machine can produce 120 toys in 40 minutes. How many toys can it produce in 2.5 hours?
A. 450 toys
B. 48 toys
C. 300 toys
D. 7.5 toys
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
$2.5 \text{ hours} \times 60 \text{ minutes/hour} = 150 \text{ minutes}$
Before setting up the proportion, ensure all time units are consistent. Convert 2.5 hours into minutes.
2
Let \(x\) be the number of toys produced in 150 minutes. $\frac{\text{Toys}}{\text{Minutes}} = \frac{120}{40} = \frac{x}{150
Now that units are consistent, set up the proportion relating toys produced to time taken.
3
$120 \times 150 = 40 \times x$ $18000 = 40x$
Cross-multiply the terms to form an equation.
4
\(x = \frac{18000}{40}\) \(x = 450\)
Divide both sides by 40 to find the total number of toys produced.

Answer: 450 toys (Option A)

Question 10Grade 6 • RP
A computer downloads a 36 MB file in 8 seconds. How long will it take to download a 90 MB file at the same rate?
A. 20 seconds
B. 11.25 seconds
C. 3240 seconds
D. 62 seconds
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let \(x\) be the time it takes to download a 90 MB file. $\frac{\text{File size (MB)}}{\text{Time (seconds)}} = \frac{36}{8} = \frac{90}{x
We set up a proportion comparing the file size to the download time, assuming a constant download rate.
2
$36 \times x = 8 \times 90$ $36x = 720$
Perform cross-multiplication to convert the proportion into a solvable linear equation.
3
\(x = \frac{720}{36}\) \(x = 20\)
Divide both sides by 36 to find the unknown download time in seconds.

Answer: 20 seconds (Option A)
