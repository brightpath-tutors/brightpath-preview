B. 13 degrees Celsius

D. 3 degrees Celsius
E. None of the above

Step-by-step solution
1
Initial temperature = -5 degrees Celsius Increase = 8 degrees Celsius
The problem describes a starting temperature and a change (increase) in temperature.
2
-5 + 8
To find the new temperature, we add the increase to the initial temperature.
3
When adding integers with different signs, subtract the absolute values and use the sign of the number with the larger absolute value. |8| - |-5| = 8 - 5 = 3 The number with the larger absolute value is 8, which is positive. So, the result is positive.
The absolute value of 8 is 8, and the absolute value of -5 is 5. Since 8 is greater than 5, we subtract 5 from 8 to get 3. The sign of 8 is positive, so the final answer is positive 3.
4
The temperature at noon was 3 degrees Celsius.
The final temperature is 3 degrees Celsius.
