Question 1Grade 7 • NS
A shopper's bank account balance changed from $150 to $120. What is the absolute value of the change in their balance?
A. $0
B. $30
C. $270
D. -$30
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Change = Final Balance - Initial Balance Change = $120 - $150 = -$30
To find the change in balance, subtract the initial balance from the final balance.
2
|-$30| = $30
The absolute value of a number is its distance from zero on the number line, which is always non-negative. So, the absolute value of -$30 is $30.

Answer: $30 (Option B)

A computer's processor speed deviates from its advertised speed. Processor X has a deviation of -0.7 GHz, and Processor Y has a deviation of 0.4 GHz. If a new processor, Processor Z, has a deviation that is the absolute value of the sum of the deviations of Processor X and Processor Y, what is the deviation of Processor Z?
A. -0.3 GHz
B. 0.4 GHz
C. 1.1 GHz
D. 0.3 GHz
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Processor X deviation = -0.7 GHz Processor Y deviation = 0.4 GHz
Extract the given deviation values from the problem statement.
2
Sum = -0.7 + 0.4 = -0.3 GHz
Add the deviation of Processor X to the deviation of Processor Y.
3
|-0.3| = 0.3 GHz
The deviation of Processor Z is the absolute value of the sum, which is 0.3 GHz.

Answer: 0.3 GHz (Option D)
