Question 1Grade 7 • NS
Maya's bank account balance is -$15. What is the absolute value of her balance?
A. -$15
B. $1
C. $15
D. $0
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Balance = -$15
The bank account balance is given as -$15, meaning she owes $15.
2
Absolute value represents distance from zero.
Absolute value is the magnitude of a number, regardless of its sign. It tells us how far a number is from zero on the number line.
3
|-$15| = $15
The absolute value of -$15 is $15, because -$15 is 15 units away from zero.

Answer: $15 (Option C)

Question 10Grade 7 • NS
Over a month, Sarah's weight changed by -3.5 pounds. John's weight changed by +2.1 pounds. What is the difference between the absolute values of their weight changes?
A. -1.4 pounds
B. 1.4 pounds
C. 5.6 pounds
D. 1.4 pounds total change
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Sarah's change = -3.5 pounds Absolute value = |-3.5| = 3.5 pounds
The absolute value of -3.5 is 3.5. This is the magnitude of Sarah's weight change.
2
John's change = +2.1 pounds Absolute value = |+2.1| = 2.1 pounds
The absolute value of +2.1 is 2.1. This is the magnitude of John's weight change.
3
Difference = 3.5 - 2.1 = 1.4 pounds
Subtract the smaller absolute value from the larger one to find the difference between their magnitudes.

Answer: 1.4 pounds (Option B)
