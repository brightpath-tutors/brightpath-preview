Question 1Grade 8 • NS
Four friends equally share a restaurant bill that has a total debt of $120. If each friend pays an equal share, what integer represents the amount, in dollars, each friend owes?
A. -116
B. 30
C. -30
D. -124
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Total debt = -$120 (since it's a debt) Number of friends = 4
A debt is represented by a negative integer. The problem states that the total debt is $120, so we represent it as -120. There are 4 friends sharing the debt.
2
-$120 / 4
To find each friend's equal share, we divide the total debt by the number of friends.
3
-120 / 4 = -30
When dividing a negative number by a positive number, the result is always negative. Therefore, each friend owes -$30.

Answer: -30 (Option C)
