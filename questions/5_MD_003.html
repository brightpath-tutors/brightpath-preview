Question 1Grade 5 • MD
Sarah bought several ribbons of different lengths for a craft project. The lengths (in yards) are: 1/2, 1/4, 1/2, 1/8, 3/4, 1/2. Which of the following descriptions accurately represents a line plot for these ribbon lengths?
A. The line plot shows 1 X above 1/8, 1 X above 1/2, 1 X above 1/4, and 1 X above 3/4.
B. The line plot shows 1 X above 1/8, 1 X above 1/4, 3 X's above 1/2, and 1 X above 3/4.
C. The line plot shows 1 X above 1/8, 2 X's above 1/4, 3 X's above 1/2, and 1 X above 3/4.
D. The line plot shows 1 X above 1/8, 1 X above 1/4, 2 X's above 1/2, and 1 X above 3/4.
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Measurements: [1/2, 1/4, 1/2, 1/8, 3/4, 1/2]
List all the given ribbon lengths.
2
Unique lengths: 1/8, 1/4, 1/2, 3/4. Count frequencies: 1/8 (1), 1/4 (1), 1/2 (3), 3/4 (1)
Identify each unique measurement and count how many times it appears in the list.
3
The smallest unit in the data is 1/8. All other fractions can be expressed as eighths: 1/4 = 2/8, 1/2 = 4/8, 3/4 = 6/8.
Determine the common denominator (8) for all fractions to ensure correct spacing on the line plot.
4
Plot description: 1 X above 1/8, 1 X above 1/4, 3 X's above 1/2, 1 X above 3/4.
Describe the line plot by stating the frequency of each unique measurement as 'X's above the corresponding fraction on the number line.

Answer: The line plot shows 1 X above 1/8, 1 X above 1/4, 3 X's above 1/2, and 1 X above 3/4. (Option B)
