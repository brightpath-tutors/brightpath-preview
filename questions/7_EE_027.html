D. 23.5 x 10^7
E. None of the above

Step-by-step solution
1
235,000,000
We need to convert the given population number into standard form (scientific notation).
2
2.35
The first non-zero digit is 2. So, 'a' will be 2.35.
3
The original number 235,000,000 has an implied decimal at the end. To get 2.35, the decimal moved 8 places to the left.
Since the original number is greater than 1, the exponent will be positive. The decimal moved 8 places, so the exponent is 8.

Question 10Grade 7 • EE
The approximate age of the universe is 13,800,000,000 years. If a single 'cosmic year' is defined as 2.25 x 10^8 years, how many cosmic years old is the universe? Express your answer in standard form.
A. 0.613 x 10^2
B. 61.3 x 10^0
C. 6.13 x 10^2
D. 6.13 x 10^1
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
13,800,000,000 = 1.38 x 10^10 years
Move the decimal 10 places to the left.
2
2.25 x 10^8 years
This is already given in standard form.
3
(1.38 x 10^10) / (2.25 x 10^8)
Divide the 'a' values and subtract the exponents.
4
1.38 / 2.25 ≈ 0.61333...
We can round 'a' to three significant figures, so 0.613.
5
10 - 8 = 2
The new exponent is 2.
6
0.613 x 10^2
This is not yet in correct standard form because 'a' (0.613) is not between 1 and 10.
7
Move the decimal point in 0.613 one place to the right to get 6.13. Since the decimal moved right, decrease the exponent by 1. 6.13 x 10^(2-1) = 6.13 x 10^1
The final answer in standard form is 6.13 x 10^1 cosmic years.

Answer: 6.13 x 10^1 (Option D)
