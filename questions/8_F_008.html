Question 1Grade 8 • F
Store A sells bags of candy for $2.50 each, plus a $3 coupon discount for any purchase. Store B sells bags of candy for $2.00 each, with no additional discounts. If a customer buys 5 bags of candy, which store offers a lower total cost?
A. They cost the same
B. Store B (Cost $10.00)
C. Store A (Cost $9.50)
D. Cannot be determined
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For Store A, the cost function is C_A(x) = 2.50x - 3. Evaluate for 5 bags: C_A(5) = 2.50(5) - 3 = 12.50 - 3 = 9.50.
Calculate the total cost for 5 bags of candy at Store A by substituting x=5 into its cost function and applying the discount.
2
For Store B, the cost function is C_B(x) = 2.00x. Evaluate for 5 bags: C_B(5) = 2.00(5) = 10.00.
Calculate the total cost for 5 bags of candy at Store B by substituting x=5 into its cost function.
3
Compare C_A(5) = $9.50 and C_B(5) = $10.00. Since $9.50 < $10.00, Store A offers a lower total cost.
Compare the calculated total costs for both stores to determine which one is lower.

Answer: Store A (Option C)

E. None of the above
🔒 Backend review only — not visible to students

3
Compare the costs: Type A costs $2550, and Type B costs $2800. Since $2800 > $2550, Type B has a higher cost for 100 square meters.
Compare the calculated total costs for both flooring types to determine which one is higher.

Question 10Grade 8 • F
A new smartphone's battery percentage, B_N(h), after h hours of use is shown in the graph below. (The graph shows a linear function with points (0,100), (5,50), (10,0).) The older model's battery percentage, B_O(h), is described as starting at 90% and decreasing by 8% every hour. Which smartphone has more remaining battery percentage after 7 hours of use?
A. Older Model (34%)
B. New Smartphone (30%)
C. They have the same remaining battery
D. Cannot be determined
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
For the new smartphone, find the slope from the graph using points (0,100) and (5,50): (50 - 100) / (5 - 0) = -50 / 5 = -10% per hour. The y-intercept is 100%. So, the function is B_N(h) = -10h + 100. Calculate battery after 7 hours: B_N(7) = -10(7) + 100 = -70 + 100 = 30%.
Derive the linear function for the new smartphone's battery life from the graph by finding its slope and y-intercept, then evaluate it for 7 hours.
2
For the older model, the initial battery is 90% and it decreases by 8% per hour. So, its function is B_O(h) = -8h + 90. Calculate battery after 7 hours: B_O(7) = -8(7) + 90 = -56 + 90 = 34%.
Formulate the linear function for the older model's battery life based on its initial percentage and hourly decrease, then evaluate it for 7 hours.
3
Compare the remaining battery percentages: New smartphone has 30%, and Older Model has 34%. Since 34% > 30%, the Older Model has more remaining battery percentage.
Compare the calculated battery percentages of both smartphones to determine which one has more remaining battery.

Answer: Older Model (Option A)
