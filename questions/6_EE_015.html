Question 1Grade 6 • EE
A store offers a discount on items costing more than $15 but less than $40. Let 'c' be the cost of an item. Write a compound inequality to represent the cost of items eligible for the discount.
A. 15 <= c <= 40
B. 15 < c < 40
C. c < 15 or c > 40
D. 15 > c > 40
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
The phrase 'more than $15' means the cost 'c' must be greater than 15, so c > 15.
This establishes the minimum value for the item's cost, not including $15 itself.
2
The phrase 'less than $40' means the cost 'c' must be less than 40, so c < 40.
This establishes the maximum value for the item's cost, not including $40 itself.
3
Since both conditions ('c > 15' AND 'c < 40') must be true for an item to be eligible, we combine them into a single compound inequality: 15 < c < 40.
For 'and' conditions where a variable is between two values, it's written with the variable in the middle and the smaller value on the left.

Answer: 15 < c < 40 (Option B)
