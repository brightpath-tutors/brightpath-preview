Question 1Grade 7 • RP
A store sells 3 identical shirts for $27. If a customer wants to buy 5 of these shirts, how much will it cost?
A. $81
B. $15
C. $9
D. $45
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Let x be the cost of 5 shirts. The proportion can be written as: 3 shirts / $27 = 5 shirts / x
We set up a proportion comparing the number of shirts to their cost. The units must be consistent on both sides of the equation.
2
3 x = 27 5 3x = 135
To solve a proportion, we cross-multiply the numerator of one ratio by the denominator of the other ratio.
3
x = 135 / 3 x = 45
Divide both sides of the equation by 3 to find the value of x, which represents the cost of 5 shirts.

Answer: $45 (Option D)

E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1

2

Question 4Grade 7 • RP

E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1

We maintain the ratio of distance to time for both trips, assuming constant speed.
2
