Question 1Grade 7 • NS
A store's account showed a balance of -$75 due to a customer return. What is the absolute value of this balance?
A. 0
B. -75
C. 75
D. 150
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Absolute value is the distance of a number from zero on a number line, always represented as a non-negative value.
The problem asks for the absolute value of -$75, which represents the magnitude of the balance regardless of whether it's a debt or credit.
2
|-75| = 75
The distance of -75 from 0 on the number line is 75 units. Therefore, the absolute value of -$75 is $75.

Answer: 75 (Option C)

Question 8Grade 7 • NS
A clock in a laboratory was initially 5 minutes ahead (+5) of standard time, but later found to be 12 minutes behind (-12) of standard time. What is the absolute value of the total change in its deviation from standard time?
A. -17
B. -7
C. 17
D. 7
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
Initial deviation = +5 minutes Final deviation = -12 minutes
The problem describes two states of the clock's deviation. We need to find the total change from the initial state to the final state.
2
Total change = Final deviation - Initial deviation Total change = -12 - (+5) = -12 - 5 = -17 minutes
To find the change, we subtract the initial deviation from the final deviation. Subtracting a positive number is the same as adding its negative.
3
|-17| = 17
The absolute value of -17 is 17. This represents the magnitude of the change in deviation, regardless of direction.

Answer: 17 (Option C)

Question 10Grade 7 • NS
A programmer adjusted a line of code, resulting in an error value of -3.12. After a second adjustment, the error value became +0.58. What is the absolute value of the difference between the two error values?
A. 3.7
B. -2.54
C. 2.54
D. -3.7
E. None of the above
🔒 Backend review only — not visible to students
Step-by-step solution
1
First error value = -3.12 Second error value = +0.58
The problem provides two distinct error values, indicating different states of the system.
2
Difference = Second error value - First error value Difference = 0.58 - (-3.12) Difference = 0.58 + 3.12 = 3.70
Subtracting a negative number is equivalent to adding its positive counterpart. This gives the total shift between the two error states.
3
|3.70| = 3.70
The absolute value of 3.70 is simply 3.70, as it is a positive number. This represents the magnitude of the difference.

Answer: 3.7 (Option A)
